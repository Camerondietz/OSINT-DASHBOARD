# OSINT-DASHBOARD

Complete dashboard to monitor activity which expands from local Austin area to entire globe.



Section	What's inside

Home	Module index, 5-level threat reference, quick emergency numbers

Radio Frequencies	60+ frequencies — scope selector (Local Austin → Texas → National → All), 11 category filters, live search. Edit RADIO\_DATA JSON to add more

Agency Directory	38 agencies — Federal, DoD/Intel, Texas, Austin Local, NGO, Cyber, International — with logos, websites, Twitter, Facebook, other links

Document Library	30 reference docs — Military FMs, ICS 100–400, NIMS, NRF, TCCC, Cyber frameworks, natural hazard guides — filterable by category

Checklists	8 interactive checklists with persistent browser storage: Bug-Out, Shelter-In-Place, Comms Plan, Vehicle Kit, Cyber Incident, Power Outage, Evacuation, Medical Emergency

Comms Plan	Editable ICS radio assignment table, contact tree, rally point planner, mesh network options (Meshtastic, APRS, Winlink)

Cybersecurity	Threat intel links, OPSEC 5-step, digital hygiene checklist, IR workflow, tools directory

Medical	TCCC MARCH protocol, training resources, emergency numbers

Supplies	6 gear lists with sortable tables: Go-Bag, Shelter, Comms, Medical, Power, Food

Maps \& GEOINT	18 curated map resources



GitHub Pages deployment — 3 files in one repo:

index.html              ← rename sentinel-hub.html to this

sentinel-dashboard.html ← the live ops dashboard

docs/                   ← drop your PDFs here

The "LIVE OPS →" FAB button on every page links back to the dashboard.
